╔══════════════════════════════════════════════════════════════════════════╗
║                                                                          ║
║                    ⚠️  WHY THE WEBSITE WON'T OPEN  ⚠️                   ║
║                                                                          ║
╚══════════════════════════════════════════════════════════════════════════╝

The error "ERR_CONNECTION_REFUSED" means the server is NOT running.

The server can't run because Node.js is NOT installed on your computer.

╔══════════════════════════════════════════════════════════════════════════╗
║                    ✅ HERE'S HOW TO FIX IT                              ║
╚══════════════════════════════════════════════════════════════════════════╝

STEP 1: INSTALL NODE.JS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

I've opened the Node.js website in your browser. If it didn't open:

   → Go to: https://nodejs.org/
   → Click the BIG GREEN button that says "LTS" (recommended)
   → Wait for download to finish

Then:
   → Open the downloaded file (node-v20.x.x-x64.msi)
   → Click "Next" → "Next" → "Next" → "Install"
   → Wait for installation
   → Click "Finish"

⚠️  IMPORTANT: RESTART YOUR COMPUTER after installation!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 2: VERIFY INSTALLATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

After restarting, double-click: check-nodejs.bat

It will tell you if Node.js is installed correctly.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 3: START THE WEBSITE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Double-click: start-website.bat

The website will open automatically at http://localhost:3000
